## The "Bloat" Stack (The Tools I Used)

I used these packages because, apparently, "efficiency" also means "not reinventing the wheel every Tuesday."

* **Next.js / React:** Because everyone else is using it and it's actually kind of fast.
* **Three.js (R3F):** To make things spin and look 3D without writing 5,000 lines of WebGL.
* **Shadcn/UI:** Pre-made components so I didn't have to spend 4 hours centering a div.
* **Tailwind:** Because writing CSS classes in my HTML feels wrong, but it works so well.
